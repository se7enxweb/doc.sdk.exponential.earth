Exponential 3 changes in 3.1

This changelog entry was not preserved in the web archive.

The 3.1 series release notes were not captured by the Wayback Machine. The page returned no archived copy at the time the snapshot was taken.

Version 3.1-1 is the release documented throughout this SDK (alias: 3.1). It follows the 2.9 series and includes the features described in the Key Features section. The most recent archived changelog entry is 2.9-7, which documents the new collaboration system, bug fixes, user interface cleanups and optimisations that formed the basis of the 3.1 release.

Known changes in the 3.1 series

  • Extensions system formalised and integrated into the kernel
  • New eZExtension class for managing active extensions
  • Module repository and extension directory settings moved to dedicated module.ini
  • Collaboration system completed and stabilised
  • Approval workflow event finalised
  • URL alias system integrated (uri_alias node attribute)
  • eZTemplate operator set expanded (arithmetic, type, control, datetime operators)
  • Translation cache system added ([RegionalSettings] TranslationCache=enabled)
  • Bork/1337 translators included for testing
  • eZXML DOM improvements: cloning, attribute removal, prefix support, namespace handling
  • Debug log rotation added to prevent log flooding
  • eZDB MySQL driver: associative-array-only fetch mode
  • eZDB PostgreSQL reconnect support
  • SDK documentation updated to cover 3.1 API surface

See 3.1-2 for the next incremental release in this series.
