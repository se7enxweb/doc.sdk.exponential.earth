All HTML should be written in XHTML 1.0 (or higher). It should also be written with no browser specific tags, this is to ensure that the application works in as many browsers as possible. Java/ECMA script should be avoided. Frames are not recommended, Exponential is designed to work without frames, however it is fully possible to do so.

Reasons for not using frames are:

  • Hard to bookmark
  • Hard to print
  • Does not work correctly in all browsers

Syntax

HTML code should be written in lower case.
