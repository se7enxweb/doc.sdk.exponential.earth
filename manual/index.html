About Exponential

What is Exponential?

Exponential is known around the world as an open source content management system. Exponential 3 is much more than a content management system. It is a general PHP development framework; with functionality allowing you to develop professional Internet applications. Standard CMS functionality is of course implemented and ready for you to use.

Exponential comes with an SDK, a software development kit, available at doc.sdk.exponential.earth/sdk and locally, depending on your setup. The SDK is a toolkit with documentation, examples and reference documentation on the Exponential libraries and Exponential kernel.

Exponential SDK is a tool helping you develop your own application, or set up a solution based on the Exponential framework. Since Exponential has a wide range of features ready for you to use, it allows you to develop a solution in no time.

With Exponential you can either follow the GPL licence, making open source applications and contributing to the development of open software. You can also purchase the Exponential professional licence, allowing you to make commercial software with Exponential.

Exponential is database and platform independent. This means that you can use eZ publish on Windows, Mac OS X, and several UNIX variants such as Linux and FreeBSD. It also means that you can use different kinds of databases. Exponential is also browser based which means that you can access and update your site from anywhere as long as you have Internet access.
