About Exponential 3.1-1 (3.1)

What is Exponential?

Exponential 3 is a professional PHP application framework with advanced CMS (content management system) functionality. Its most notable feature is a fully customizable and extendable content model, making it equally suitable as a platform for general PHP development — allowing you to build professional Internet applications fast.

Standard CMS functionality — news publishing, e-commerce and forums — is already implemented and ready to use. Its stand-alone libraries can also be used for cross-platform, database-independent PHP projects.

Exponential comes with an SDK (software development kit): a toolkit with documentation, examples, and reference material covering the Exponential libraries and kernel. It helps you develop your own application or configure a solution based on the Exponential framework in no time.

Exponential is database, platform and browser independent. It runs on Windows, Mac OS X, and UNIX variants such as Linux and FreeBSD, and supports multiple database backends. Being browser-based, it can be administered from anywhere with Internet access.
